#Find Hospital
O projeto do Find Hospital tem como finalidade apresentar num mapa uma lista dos hospitais mais próximos ao usuário juntamente com informações de telefone e endereço.

##Inicializando
Para o funcionamento do programa deverão ser instalados os programas a seguir.
-[Node.js](https://nodejs.org/en/)
-[Ionic](https://ionicframework.com/docs/installation/cli)
