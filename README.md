# Find Hospital
O projeto do Find Hospital tem como finalidade apresentar num mapa uma lista dos hospitais mais próximos ao usuário juntamente com informações de telefone e endereço.</br>

## Inicializando
Para o funcionamento do programa deverão ser instalados os programas a seguir.</br>
-[Node.js](https://nodejs.org/en/)</br>
-[Ionic](https://ionicframework.com/docs/installation/cli)</br>
</br>
Na pasta onde se encontram os arquivos do programa deve ser efetuado o comando ionic serve, que irá abrir o endereço local de funcionamento do programa.

## Tutorial
Para o desenvolvimento do programa foi realizado o tutorial de [Ionic/Angular](https://ionicframework.com/docs/v3/intro/tutorial/).</br>
